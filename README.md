A simple ASP.NET Core Web API that simulates basic banking operations such as account management and transactions. Built for learning backend concepts like REST APIs, DTOs, services, and database integration using Entity Framework Core.

Features

-Create and manage bank accounts

-Perform basic transactions (deposit, withdraw, transfer)

-SQLite database with Entity Framework Core

-Clean separation using Controllers, Services, Models, and DTOs

***Tech Stack***

*ASP.NET Core

*C#

*Entity Framework Core

*SQLite
